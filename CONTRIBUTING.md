# Contributing

Thank you for your interest in WaffleX.

## Current Status

WaffleX is currently in an early pre-release stage. The public repository exists primarily as a project landing page while research, design, and implementation planning continue.

Because of that, external contributions are not open yet.

## What to Expect Later

Once the project is ready for broader collaboration, this document will be expanded to include:

- contribution workflow
- branching strategy
- coding standards
- issue triage process
- review and acceptance criteria
- responsible research expectations

## For Now

If you are interested in collaboration, conference discussion, or future contribution opportunities, please watch the repository for updates.
