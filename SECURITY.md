# Security Policy

## Project Status

WaffleX is currently in development and has not yet been publicly released.

## Reporting a Security Issue

If you believe you have identified a security issue related to this repository or any future release artifacts, please do not disclose it publicly before coordinated review.

Please contact the maintainer privately once contact details are published in this repository.

## Scope

At this stage, the repository may not contain functional tooling. Security reporting procedures will be expanded as the project matures.
